# 📘 GenericBuddy User Manual  
**Platform:** [genericbuddy.streamlit.app](https://genericbuddy.streamlit.app)  
**Date:** July 2025  

---

## 🧩 Overview

**GenericBuddy** is a healthcare tool designed to help users:
- 💊 Find affordable **generic medicine alternatives**.
- 🏥 Locate nearby **pharmacies** using various filters.
- 📄 Upload **prescriptions** to automatically extract and identify medicines.

Its mission is to make healthcare more affordable and accessible by guiding users toward cost-effective generic drugs and accessible pharmacy services.

---

## 🚀 Main Features

### 🔹 Home Page
- Displays app branding and tagline.
- Promotes saving money on healthcare.
- Provides a sidebar for easy navigation and theme toggle (Light/Dark mode).

---

### 🔹 1. Generic Medicine Finder

Use this feature to:
- Search for branded or generic medicines.
- Compare prices and find alternatives.
- Filter by type, formulation, and dosage.

#### 🧮 Filters Available:
- **Name**: Search by medicine name.
- **Formulation**: Tablet, syrup, etc.
- **Ailment**: e.g., pain, fever, diabetes.
- **Therapeutic Type**
- **Dosage Filter**
- **Formulation Type**
- **Branded Medicine Name**
- **Sort By**: Generic price
- **Order**: Low → High or High → Low

#### 📊 Results Displayed:
- Generic and branded names.
- Dosage information.
- Prices of generic and branded alternatives.
- Percentage savings.

💡 *Example: Searching “Fever” shows various Paracetamol-based formulations with respective prices and savings.*

---

### 🔹 2. Pharmacy Locator

This tool helps you find nearby pharmacies using:
- **PIN Code**
- **Area / Locality**
- **City**
- **Current Location (GPS)**

📏 You can set a **search radius (1 to 20 km)** for more accurate results.

---

### 🔹 3. Prescription Reader

Allows users to:
- Upload **PDF or PNG** prescriptions.
- Extract medicine names using OCR.
- Auto-suggest generic alternatives.

📁 **File Limits**:
- Format: PNG or PDF  
- Size: Up to 200 MB per file

## 💡 Tips for Best Use

- Use broad search terms for ailments (e.g., “pain” instead of “back pain”).
- Start with low radius in Pharmacy Locator and increase if needed.
- Check all dosage options to find the best price-to-effectiveness ratio.
- Use the Prescription Reader for quick identification instead of manual entry.
