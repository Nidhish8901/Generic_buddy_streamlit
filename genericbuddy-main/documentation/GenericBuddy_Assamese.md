# 📘 প্ৰকল্প নথি: GenericBuddy – ঔষধ সন্ধান আৰু ফাৰ্মাচী লোকেটৰ

## 📌 প্ৰকল্প শিৰোনাম
**GenericBuddy – ঔষধ সন্ধান আৰু ফাৰ্মাচী লোকেটৰ**

## 🧠 উদ্দেশ্য
- সহজ আৰু কম খৰচৰ মাধ্যমত ঔষধ চাব
- ডোজ, ফৰ্মুলেশ্বন, মূল্য সঞ্চয় পৰীক্ষা
- লোকেশ্বন অনুসৰি ফাৰ্মাচী বিচৰা

## 🔍 সমস্যাৰ বিৱৰণ
ভাৰতত ব্ৰেণ্ডেড ঔষধৰ দাম অধিক। বহুতে জেনেৰিক বিকল্প জানে না। এই প্লাটফৰ্মে এনে সমস্যা সমাধান কৰে।

## 🧩 মুখ্য উপাদান

### 1️⃣ ডাটা সংগ্ৰহ
- Excel database: নাম, ডোজ, মূল্য, সঞ্চয়

### 2️⃣ ঔষধ বিকল্প
- AI সহায়তা
- নাম / ফৰ্মুলেশ্বন মিল

### 3️⃣ Web App বিকাশ

### 4️⃣ Features

#### 💊 Medicine Finder
- নাম অনুসন্ধান, Filter, ডোজ, side effects

#### 🗺️ Pharmacy Locator
- PIN, নগৰ, লাইভ GPS
- radius স্লাইডাৰ, Export, Map view
